Dex

Purpose: The main database object that is used in almost every call. In most cases, there should be only 1 instance of the Dex class in any single piece of code

Constructor

new Dex([url], [allowReconnect])

Parameters:
Name Type Attributes Default Description
url string
optional
 ""

Indicates the database address to connect to If this parameter is provided, the driver will automatically begin the connection process If not, then the user is required to use Dex.connect when it is time to connect.
allowReconnect boolean
optional
 true

Indicates if the driver should try to reconnect to the database after the websocket is unintentionally closed

close()

Purpose: Manually disconnect from the database and close websocket connection

Note: This function overrides the allowReconnect setting and will prevent the websocket from reestablishing a connection. To reconnect after running this function, use Dex.connect

connect(url)

Purpose: Begin connection with database

Note: This function is unnecessary if a url was passed to the Dex object when constructing it. This function is only necessary if no url was passed or if the websocket connection has been manually closed with Dex.close

Parameters:
Name Type Description
url string

Indicates the database address to connect to
