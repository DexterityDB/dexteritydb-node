ReadOp

ReadOp

Purpose: An abstract class that represents a single operator that can be interpreted by the driver. A ReadOp is created to translate different combinations of patterns into something that the database can understand.

This class is created internally as a result of other methods and should never be constructed directly.

Constructor

new ReadOp()

Should not be called by the user
