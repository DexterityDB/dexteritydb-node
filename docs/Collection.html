Collection

Collection

Purpose: A Collection object based on the collection that is passed to its constructor

This class is created internally as a result of other methods and should never be constructed directly.

Constructor

new Collection()

Should not be called by the user

Members

name

Purpose: Returns the name of the collection

Methods

drop() → {Promise}

Purpose: Drop or remove a collection (and its contents) from the database

Example:

await collection.drop();
Returns:

true if collection was dropped, false if unsuccessful

Type
Promise

explain(isOn) → {Collection}

Purpose: Set a parameter on the Collection object that tells the database to provide information about the queries that are performed

Example:

collection.explain().find({ name: "Alex" }).fetchAll().then((results, t) => {
 console.log(results);
 console.log(t);
});
Parameters:
Name Type Default Description
isOn boolean true

Indicates if parameter should be set or unset
Returns:

The same Collection that called the function, with the modified parameter

Type
Collection

find(pattern) → {ReadQuery}

Purpose: Find a specified pattern in the database. Can be chained with a ReadQuery consumable method

Example:

collection.find({ name: "Dillon" }).fetchAll().then((results, t) => {
 console.log(results);
 console.log(t);
});
Parameters:
Name Type Description
pattern ReadOp | JSON | null

The pattern that is being searched for; passing null will return everything in the collection
Returns:

A new ReadQuery that contains the pattern to be searched

Type
ReadQuery

index(indexName) → {Promise}

Purpose: Index a field. This allows the field to be searchable

Example:

await collection.index("name");
Parameters:
Name Type Description
indexName string

The name of the field to be indexed
Returns:

true if the field was indexed or if it has previously been indexed, false if unsuccessful

Type
Promise

insert(…items) → {Promise}

Purpose: Monolithic method for inserting one or more items into the collection. Item should be in JSON format or an array of JSON objects

Example:

await collection.insert({ name: "Tom", position: "marketing" }, { name: "Todd", position: "sales"});
Parameters:
Name Type Attributes Description
items JSON
repeatable

One or more items that should be added to the collection. Can be individual items or an array of items
Returns:

true if the item(s) is successfully inserted, false if the insert fails

Type
Promise

options(options) → {Collection}

Purpose: Set options for a query

Example:

collection.options({ explain: true });

Note: Options are being worked on. There will be more in the future...

Parameters:
Name Type Description
options JSON

A field-value pair that contains one or more options and their desired values
↳ explain boolean

Sets a parameter on the Collection object that tells the database to provide information about the queries that are performed - same functionality as Collection.explain
Returns:

A new Collection with the desired options set

Type
Collection

remove(pattern) → {Promise}

Purpose: Monolithic method for removing one or more items in the collection based on a pattern

Example:

await collection.remove("name");
Parameters:
Name Type Description
pattern ReadOp | JSON | null

The pattern that matches the items that should be removed; passing a null will remove all items in the collection
Returns:

true if the item(s) are successfully removedd, false if the removal fails

Type
Promise

removeIndex(indexName) → {Promise}

Purpose: Removes the index on a field in the collection

Example:

await collection.removeIndex("name");
Parameters:
Name Type Description
indexName string

The name of the field that should be unindexed
Returns:

true if the index was successfully removed, false if the removal fails

Type
Promise

replace(pattern, item) → {Promise}

Purpose: Monolithic method for replacing one or more items with a new item

Example:

await collection.replace({ name: "Todd" }, { name: "Tom", age: 23 });
Parameters:
Name Type Description
pattern ReadOp | JSON | null

The pattern to match the items that will be replaced
item JSON

The item that will be replacing the matched items
Returns:

true if the replacement was successful, false if the replacement fails

Type
Promise

update(pattern, updateFields) → {Promise}

Purpose: Monolithic method for updating one or more items by changing the value of one or more fields or completely deleting a field or fields from the items

Example:

await collection.update({ name: "Tom" }), { name: "Todd", age: Dex.delete() });
Parameters:
Name Type Description
pattern ReadOp | JSON | null

The pattern to match the items that will be updated
updateFields JSON

Field-value pairs that indicate what the new value of the field(s) should be or a PartialDelete shorthand operator
Returns:

true if the update was successful, false if the update fails

Type
Promise
